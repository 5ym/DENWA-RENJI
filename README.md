# DENWA-RENJI

電話レンジ(仮)の(仮)

## feature

電話レンジ(仮)を再現するためにスマホ等でイヤホンジャックよりarduinoを接続し受電すると音声が自動で流し、発信側でプッシュトーンを送るとarduino側で認識できるようにしたもの
